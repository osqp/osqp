Linear Systems Solvers TypesÔÉÅ

The settings parameter osqp_linsys_solver_type defines the solver type for the linear system. This is an enum type whose options are OSQP_UNKNOWN_SOLVER, OSQP_DIRECT_SOLVER, and OSQP_INDIRECT_SOLVER.
