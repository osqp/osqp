Algebra Backends

OSQP supports several algebra backends that can be used to create and solve QP problems. Currently, these are builtin, MKL, and CUDA. The builtin backend is the default, and always available, regardless of whether you’re using the C interface directly, or any of the language bindings.

The solver types osqp_linsys_solver_type that are available to you depend on the backend that you’re using. The following table lists the solver types available for each backend:
